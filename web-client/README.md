#GT Thrift Shop - Web Client

### Setup

#### Dependencies

* [Node.js](http://nodejs.org)
* [npm](https://www.npmjs.org)

#### Installation

```sh
$ npm install -g gulp bower
$ npm install
$ bower install
```

#### Application Start

```sh
$ gulp debug
```

Point browser to [http://localhost:8000](http://localhost:8000)
